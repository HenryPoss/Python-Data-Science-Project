# Python-Data-Science-Project

This project follows the data science process on the 'penguins' dataset (added to repository).

The following was performed in this project:

- Performed Exploratory Data Analysis (EDA) - Discovery, Cleaning, Validation, Structuring.
- Data Visualisations to explore attributes and relationships.
- Simple Linear Regression model to predict a variable given another variabel (including checking model assumptions).
- K-Means Clustering model to organise the data into distinct groups.
- Random Forest Classification model to predict a feature of the dataset given a series of variables.
- Optimisation of models to improve performance.
- Evaluation of models to understand performance.
- List of insights discovered from exploratory data analysis and model creation.
